# OpenAI Model Options and Intelligence Levels

This repository contains the single-file HTML guide `OpenAI Model Options and Intelligence Levels.html`.

The guide is a truth-first, responsive, interactive reference page for model option and intelligence-level selection.

RC1 hardening has been completed and merged into `main`.

Public-safe agent instruction files are included:

- `GEMINI.md`
- `AGENTS.md`

Runtime smoke testing was performed locally through `http://localhost:8080/`, and Chrome console checks were clean during the final check.
